# EMHentai


## Abstract

A simple, pure Swift, easy to use, EHentai Viewer For iOS

一个简单，纯Swift，容易使用的Ehentai的iOS APP



## Install

Download ZIP / git clone

cd EMHentai

pod install



## Feature

- [x] 查看Ehentai上的本子
- [x] 高级搜索
- [x] 历史记录
- [x] 下载
- [x] 相关Tag搜索
- [x] Tag翻译
- [x] E-hentai/ExHentai切换
- [ ] 多语言支持
