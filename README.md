# EMHentai



## 简介

一个简单，纯Swift，容易使用的Ehentai的iOS APP



## 功能

- [x] 查看Ehentai上的本子
- [x] 高级搜索
- [x] 历史记录
- [x] 下载
- [x] 相关Tag搜索
- [x] Tag翻译
- [x] E-hentai/ExHentai切换
- [x] 适配iPad
- [x] 适配暗黑模式
- [x] 适配横竖屏



## 安装

iOS版本要求：>= 13.0

因为考虑到Ehentai的法律风险，这里就不提供直接的IPA安装包了。

有Mac的同学可以直接使用Xcode安装（无需开发者账号）：

Download ZIP / git clone

cd EMHentai

pod install
